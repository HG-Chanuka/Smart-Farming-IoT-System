Wireless Smart Farming & IoT Irrigation System 🌿
An automated, wireless irrigation and monitoring system designed to optimize water usage and monitor crop health in real-time.

🚀 Key Features
Wireless Connectivity: Uses ESP-NOW protocol for high-speed, low-power communication between sensor nodes.

Real-time Monitoring: Tracks Soil Moisture, Temperature, and Humidity using DHT11 and Soil sensors.

Automated Irrigation: Automatically triggers water pumps based on sensor thresholds.

Dual-Node Architecture: Features a multi-node system (Node A & Node B) for wider field coverage.

🛠️ Hardware Used
Microcontrollers: ESP32 (Master Node), Arduino Uno (Sensor Node).

Sensors: DHT11 (Temp/Humidity), Soil Moisture Sensors, PIR Motion Sensor.

💻 Tech Stack
Language: C++ / Arduino Programming.

Tools: Arduino IDE, GitHub Desktop.
